# Events project

File based routing

### Planning

| Path                | Action                                      |
| ------------------- | ------------------------------------------- |
| `/`                 | Starting Page (show featured events)        |
| `/events`           | Events Page (Show all events)               |
| `/events/<some-id>` | Event detail page (show selected event)     |
| `/events/...slug`   | Filtered events page (show filtered events) |
